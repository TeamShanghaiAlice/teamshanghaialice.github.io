# Django 1.4 alpha 发布了，相关特性介绍 #



	 -

		Django 1.4 now requires a minimum of Python 2.5; Python 2.4 不在被支持.



	 -

		Django终于要支持html5了！在Django1.4中，其admin模板中将使用html5文档类型了。



	 -

		select_for_update()  对事务支持，不过sqlite3不支持这一特性。



	 -

		1.4支持Selenium



	 -

		tag支持关键字和非关键字参数了



	 -

		不见nosql的支持
